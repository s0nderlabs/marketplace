# s0nderlabs marketplace

Plugin marketplace for Claude Code.

## Install

```bash
/plugin marketplace add s0nderlabs/s0nderlabs-marketplace
```

## Available plugins

| Plugin | Description |
|--------|-------------|
| [attn](https://github.com/s0nderlabs/attn) | Agent-to-agent encrypted messaging |

## Install a plugin

```bash
/plugin install attn@s0nderlabs
```
